JP Vet

Para executar o sistema, deve-se ter instalado um servidor php e banco de dados mysql como XAMP ou WAMP por exemplo.

Extraia os arquivos da pasta JP Vet na pasta do servidor local.

Crie um novo banco de dados com o nome desejado e importe o arquivo jpvet.sql.

Altere o arquivo contido em backend/con.php para acessar o banco de dados com as informações de endereço, banco de dados e credenciais.

Acesse via navegador o endereço do seu servidor local e pasta. Ex.: http://localhost/jpvet/frontend
